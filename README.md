## 📌 Proje Hakkında
Bu proje, bilgisayar bilimlerinin temel bileşenlerini anlamak amacıyla hazırlanmıştır.  
CPU, RAM ve depolama birimlerinin görevleri ve birbirleriyle olan ilişkileri açıklanmıştır.

---

## 🧠 CPU (Merkezi İşlem Birimi)

> Bilgisayarın **beyni** olarak görev yapar.

- Komutları işler  
- Hesaplamaları yapar  
- Sistemi kontrol eder  

⚡ **Performansa etkisi:**  
CPU ne kadar güçlü olursa bilgisayar o kadar hızlı çalışır.

---

## 💾 RAM (Bellek)

> Geçici veri depolama alanıdır.

- Aktif programları tutar  
- Hızlı erişim sağlar  
- Bilgisayar kapanınca veriler silinir  

🚀 **Avantaj:**  
Daha fazla RAM = Daha akıcı kullanım

---

## 🗄️ Depolama (Disk)

> Verilerin kalıcı olarak saklandığı yer.

### 🔹 HDD
- Daha yavaş
- Ucuz

### 🔹 SSD
- Çok hızlı ⚡
- Daha dayanıklı
- Günümüzde tercih edilir
